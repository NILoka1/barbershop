# 🏠 Главная

Добро пожаловать в документацию проекта **Barbershop** — полного решения для управления записями в барбершопе.

Этот проект — **монорепозиторий** на базе `pnpm`, включающий:

- 🖥️ **Фронтенд** (`client`) — React + TypeScript + Vite + Mantine
- ⚙️ **Бэкенд** (`server`) — Express + tRPC + Prisma + PostgreSQL
- 🔗 **Общие типы** (`shared`) — для типобезопасного взаимодействия
