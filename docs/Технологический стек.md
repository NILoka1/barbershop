# ⚙️ Технологический стек

## 🖥️ Фронтенд (`client`)

| Компонент          | Технология                   | Назначение |
|--------------------|------------------------------|-----------|
| Фреймворк          | React 19                     | UI-компоненты |
| Сборка             | Vite                         | Быстрая разработка |
| UI-библиотека      | Mantine (v9)                 | Готовые компоненты, темы |
| Маршрутизация      | react-router-dom@7           | Навигация |
| Управление состоянием | Zustand                    | Глобальное состояние (авторизация, модалки) |
| Запросы к API      | @tanstack/react-query        | Кеширование, фоновое обновление |
| tRPC               | @trpc/react-query            | Типобезопасные вызовы API |
| Валидация          | zod (через `shared`)         | Проверка форм |
| Линтинг            | eslint + typescript-eslint   | Качество кода |
| Стили              | CSS-in-JS (через Mantine) + tailwind-merge | Управление классами |

## ⚙️ Бэкенд (`server`)

| Компонент          | Технология                   | Назначение |
|--------------------|------------------------------|-----------|
| Сервер             | Express.js                   | HTTP-сервер |
| API                | tRPC (AppRouter)             | Типобезопасные endpoint'ы |
| База данных        | PostgreSQL (16)              | Хранение данных |
| ORM                | Prisma                       | Работа с БД, миграции |
| Аутентификация     | JWT + bcryptjs               | Авторизация пользователей |
| Валидация          | zod                          | Проверка входных данных |
| Переменные         | dotenv                       | Управление окружением |
| CORS               | cors                         | Разрешение запросов с фронтенда |

## 🔗 Общие (`shared`)

| Компонент          | Технология                   | Назначение |
|--------------------|------------------------------|-----------|
| Типы               | TypeScript                   | Общие интерфейсы |
| Валидация          | zod                          | Общие схемы валидации |
| Экспорт            | `exports` в package.json     | Импорт через `shared` |

## 🐳 Инфраструктура

| Компонент          | Технология                   | Назначение |
|--------------------|------------------------------|-----------|
| Контейнеризация    | Docker + docker-compose      | Запуск БД и сидов |
| БД                 | PostgreSQL                   | Хранение данных |
| Управление пакетами | pnpm workspace               | Монорепозиторий |
| Git-хуки           | husky + commitlint           | Чистые коммиты |

> 💡 Все зависимости типобезопасны: tRPC + Zod обеспечивают сквозную проверку типов от клиента до БД.
