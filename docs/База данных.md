# 🗄️ База данных

Проект использует **PostgreSQL** через **Prisma ORM**.

## 📂 Схема БД

Схема определена в `prisma/schema.prisma`.

### Сущности

```prisma
model Client {
  id        String   @id @default(cuid())
  email     String   @unique
  password  String
  name      String
  phone     String?
  bookings  Booking[]
}

model Worker {
  id         String   @id @default(cuid())
  email      String   @unique
  password   String
  name       String
  phone      String?
  isAdmin    Boolean  @default(false)
  shifts     Shift[]
  workerServices WorkerService[]
}

model Service {
  id           String      @id @default(cuid())
  name         String
  category     ServiceCategory
  duration     Int
  price        Int 
  workerServices WorkerService[]
  bookings     Booking[]
}

model Shift {
  id        String   @id @default(cuid())
  workerId  String
  worker    Worker   @relation(fields: [workerId], references: [id], onDelete: Cascade)
  startTime DateTime
  endTime   DateTime
  bookings  Booking[]
}

model Booking {
  id        String        @id @default(cuid())
  clientId  String
  client    Client        @relation(fields: [clientId], references: [id], onDelete: Cascade)
  serviceId String
  service   Service       @relation(fields: [serviceId], references: [id])
  shiftId   String
  shift     Shift         @relation(fields: [shiftId], references: [id], onDelete: Cascade)
  startTime DateTime
  endTime   DateTime
  status    BookingStatus @default(PENDING)
}

enum ServiceCategory {
  SALON
  MANICURE
  COSMETICS
}

enum BookingStatus {
  PENDING
  CONFIRMED
  CANCELLED
  COMPLETED
  NOSHOW
}
```

## 🔗 Связи

- `Worker` → `Shift`: один ко многим
- `Worker` → `Service` через `WorkerService` (многие ко многим)
- `Shift` → `Booking`: один ко многим
- `Client` → `Booking`: один ко многим
- `Service` → `Booking`: один ко многим

## 🛠️ Миграции

- Миграции управляются через `prisma migrate dev`
- Сиды: `pnpm run seed`
- Prisma Studio: `pnpm run prisma:studio` — визуальный редактор БД

## 🔐 Безопасность

- Пароли хранятся в виде хешей (`bcryptjs`)
- JWT используется для сессий

> 💡 Совет: Регулярно запускайте `prisma format`, чтобы схема была читаемой.
